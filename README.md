## circle-menu

## Requirements

- iOS 8.0+
- Xcode 7.2

## Installation

Just add CircleMenuLib folder to your project.

or use [CocoaPods](https://cocoapods.org) with Podfile:
``` ruby
pod 'CircleMenu', '~> 0.0.1'
```
